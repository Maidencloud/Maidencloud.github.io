Comment trouver un code y-fix

L'inconvéniant du code 60 hz c'est qu'il décale fortement l'image sur l'axe verticale (décalage vers le haut), il faut donc un code que l'on appelle y-fix, pour recentrer plus ou moins l'image. Tout comme le code 60 hz il vous faudra utilise le programme ps2dis. Je vais vous montrer comment j'ai procédé pour final fantasy 7.

Premiere chose à faire :

- Faire apparraître les fichiers caché

- Dans explorer cliquer en haut, sur outils puis sur options des dossier puis sur l'onglet affichage dans la fenêtre, descendre jusqu'a atteindre Fichier et dossiers cachés juste en dessous cliquer sur Afficher les fichiers et dossier cachés
- Faites ensuite OK

Deuxieme chose à faire :

- Insérez le cd du jeu dans l'un de vos lecteurs CD/DVD

Troisieme chose à faire

- Téléchargez le programme ps2dis

C'est bon vous êtes prêt

Comment utiliser ps2dis pour rechercher le code y-fix

Il a 2 type de y-fix, il y a donc deux code pour les trouver. Dans certain jeux c'est le type 1 dans d'autre cas le type 2. Dans le cas de final fantasy 7 c'est le type 2 que nous allons utiliser.

- Apres avoir télécharger ps2dis dézipper le
- Ouvrez le dossier dans lequel vous avez dézippé ps2dis - Puis Démarrer le programme
- faire glisser les deux fichier CMPLabelXxX.pis CMPLabelXxX2.pis dans la fenêtre ouverte de ps2dis
- Dans le programme, cliquez dans file puis open un explorateur s'ouvre. cliquer sur le lecteur ou vous avez inséré le jeux (ex : D) . A la racine du cd se trouve un fichier nommé généralement SCES_xxx.xx ou SLES_xxx.xx ou autres (SCUS, SLUS etant si je ne me trompe pas, les version americaine des fichiers, pour console americaine)
- Double cliquez dessus
Vous voilà revenus dans la fenêtre principale de ps2dis
- Cliquez sur edit, puis find pattern, puis cliquez sur as hex string puis entrez le code pattern suivant 13 00 91 24 10 00 91 24
- Le programme devrait trouver quelque chose qui ressemble à dans le cas de Final fantasy 7 a ce qui suit :

CODE Y-fix TYPE 2

000444d4 24910013 addiu s1, a0, $0013
000444d8 24910010 addiu s1, a0, $0010 <-- y-fix
(rappel du code pattern entrée plus haut)

Code pattern = 13 00 91 24 10 00 91 24

- On remplace le premier chiffre de 000444d8 par un 8 et on garde lors de l'utilisation du code dans l'action replay 0000

Ce qui nous donne 800444d8-0000

Il ne vous restes plus qu'à l'entrée dans l'action replay de votre choix.

CODE y-fix TYPE 1

24900013 addiu s0, a0, $0013
24900010 addiu s0, a0, $0010 <-- code y-fix
8622000e lh v0, $000e(s1)

Code Pattern = 13 00 90 24 10 00 90 24

Le principe est le même, il faut juste remlacer le code pattern inscrit dans l'exemple pour final fantasy 7 plus haut

Sur d'autre jeux que celui de mon exemple, le but et de chercher quel type de y-fix et present dans le code. Il faut donc chercher avec les deux code pattern . Si en essayant le premier vous ne trouver pas les formes de code ecrit en jaune. Alors il y a de forte chances que cela soit le second type.

Si malgré ça, ça ne fontionne pas, c'est que le jeu à peut être une protection "anti codes action replay" ou alors que votre action replay n'est pas assez recent (ce qui rejoint finalement l'hypothese de la protecion), soit le code pattern pour trouver le code y-fix n'est pas compatible avec ce jeu (hypothese), soit vous vous êtes trompé, soit j'en sais rien.
