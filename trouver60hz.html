Comment trouver un code 60 hz NTSC pour son jeu pal

Pour trouver le code 60 hz de son jeu il faut utiliser le programme PS2dis et insérer le CD du jeu pal dont vous voulez le code 60 hz

Premiere chose à faire :

- Faire apparraître les fichiers caché

- Dans explorer cliquer en haut, sur outils puis sur options des dossier puis sur l'onglet affichage dans la fenêtre, descendre jusqu'a atteindre Fichier et dossiers cachés juste en dessous cliquer sur Afficher les fichiers et dossier cachés
- Faites ensuite OK

Deuxieme chose à faire :

- Insérez le cd du jeu dans l'un de vos lecteurs CD/DVD

Troisieme chose à faire

- Téléchargez le programme ps2dis

C'est bon vous êtes prêt

Comment utiliser ps2dis pour rechercher le code 60 hz et y-fix (cela ne fontionne pas sur ps2, en tout cas non pucé)

- Apres l'avoir télécharger dézipper le
- Ouvrez le dossier dans lequel vous avez dézippé ps2dis - Puis Démarrer le programme
- faire glisser les deux fichier CMPLabelXxX.pis CMPLabelXxX2.pis dans la fenêtre ouverte de ps2dis
- Dans le programme, cliquez dans file puis open un explorateur s'ouvre. cliquer sur le lecteur ou vous avez inséré le jeux (ex : D) . A la racine du cd se trouve un fichier nommé généralement SCES_xxx.xx ou SLES_xxx.xx ou autres (SCUS, SLUS etant si je ne me trompe pas, les version americaine des fichiers, pour console americaine)
- Double cliquez dessus
Vous voilà revenus dans la fenêtre principale de ps2dis
- Cliquez sur edit, puis find pattern, puis cliquez sur as hex string puis entrez le code pattern suivant 80 10 80 1f
- Le programme devrait trouver quelque chose qui ressemble à, dans le cas de Final fantasy 7 pal à :

00051548 1f801080 bgtz gp, $0005574c
0005154c 00000000 nop <-------------- code 60 hz

- On remplace le premier chiffre de 0005154c par un 8 et on garde lors de l'utilisation du code dans l'action replay 0000

Ce qui nous donne 8005154C-0000

Il ne vous restes plus qu'à l'entrée dans l'action replay de votre choix.

Cas particulier

- Je me suis décidé a vérifier, ce que je disais, et je dois dire que ça marche pas toujours comme je l'ai indiqué.

- Prenons le cas de Fnal Fantasy 9 pal

Quand on tape le code pattern 80 10 80 1f on tombe sur ça

00067ff8 1f801080 bgtz gp, $0006clfc (+4225 )
00067ffc 00000000 nop
00068000 00000000 nop
00068004 00000000 nop <--- C'est la ligne où se trouve le code 60 hz

Comme vous pouvez le constater, le code 60 hz ne se trouve pas au même endroit que pour final fantasy 7. Il se trouve exactement 2 ligne plus bas.

- Prenons le cas de final fantasy 8 pal maintenant, que je vais verifier en même temps que vous.

Même code pattern 80 10 80 1f , cela nous donne 2 solutions. En appuyant sur F5 on peut voir que le code pattern est present a deux endroit dans le code. Donc il faut choisir celui qui se rapproche le plus des premiere forme qui sont présenté sur cette page , ce qui nous donne cette forme là

0005dac4 1f801080 bgtz gp, $00061cc8 (+4225 )
0005dac8 00000000 nop
0005dacc 00000000 nop
0005dad0 00000000 nop <--- C'est la ligne où se trouve le code 60 hz

Comme vous pouvez le constatez, le code 60 hz ne se trouve pas au même endroit que pour l'exemple de Final Fantasy 7, mais en revanche il se trouve au même endroit que celui de Final Fantasy 9. A ceci prêt qu'il faille user de la touche F5 pour le trouver parmi deux choix.
Il est intérrésssant de noté que le chiffre +4225 à côté du triangle inversé et sur la ligne du code pattern, se retrouve aussi bien dans final fantasy 8 que 9. Indice pour trouver d'autre code 60 hz sur d'autre jeux ? Je ne sais pas ! En revanche vous comprendrez qu'il faille faire un minimum de recherche et de test.
